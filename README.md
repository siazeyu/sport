# 工程简介
这是一个SpringBoot开发模板，也可用于SpringCloud模板开发， 目前模板包括


+ 默认包config、controller、mapper、entity、service、util、annotation
+ 统一返回数据和异常处理
+ 自定义日志注解
+ Swagger模板
+ Mybatis-plus模板
+ Dubbo模板


# 即将完成的功能

+ 用户登录模板（JWT）
+ 常用工具类的完善
+ 基本线程池模板
+ MQ模板
+ Redis模板
+ 优雅启动/停止服务模板
+ 更多功能请关注
